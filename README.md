# Cyberpunk RED Support

Projeto React + TypeScript + Vite para apoio a mesas de Cyberpunk RED.

## Como rodar

```bash
npm install
cp .env.example .env
npm run dev
```

## Stack

- React
- TypeScript
- Vite
- React Router
- Axios
- Zustand
- React Hook Form
- Zod

## Estrutura

- `design/` UI, tema e layout
- `scripts/` stores, hooks, guards e utilitários
- `integrations/` integrações com APIs
- `pages/` páginas do app
